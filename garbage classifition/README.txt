
📦 Waste Classifier - 30% Complete Project

✅ Features Implemented:
- Upload image
- Predict waste type using pre-trained Keras model
- Streamlit interface
- Displays confidence and basic SDG goal image
- Error-free and functional UI

🚀 How to Run:
1. Make sure you have Python 3.7+ installed
2. Install required packages using:
   pip install -r requirements.txt

3. Run the app:
   streamlit run app.py

📁 Required Files:
- keras_model.h5 (included)
- labels.txt (included)
- .env file (optional for OpenAI API, not needed for current version)

📝 Notes:
- No database connection needed at this stage
- All files are self-contained for submission
